HTML Link Element Demo

Example 1 - rel/type/href

<link rel="stylesheet" type="text/css" href="styles/main.css">

The link element must contain both the rel and href attributes in order to be a valid element.

Example 2 - crossorigin

<link href="https://somewebsite.com/song.mp3" crossorigin="use-credentials">

In example 2 here, somewebsite.com has a CORS policy in place on song.mp3.
Since my browser has permission to access this file via a cookie, I can specify "use-credentials" so my browser knows to send the cookie when making the file request.

Example 3 - rev

<link rel="stylesheet" rev="author" type="text/css" href="styles/main.css">

In example 3 here, I am specifying that I am the author of the linked CSS file.

Example 4 - media

<link rel="stylesheet" type="text/css" media="print" href="styles/main.css">

In example 4 here, this CSS sheet only applies to printers so the page can look nice when printed.

Example 5 - nonce

<link rel="script" nonce="abcde12345" href="styles/main.css">

In example 5 here, the file can only be linked if it has the matching nonce attribute value.

Example 6 - hreflang

<link rel="alternate" hreflang="es" href="https://es.mysite.com">

Here I link to an alternate version of my website that has been translate to Spanish, this would help with SEO in spanish speaking countries.

Example 7 - referrerpolicy

<link rel="stylesheet" referrerpolicy="origin" href="https://othersite.com/style.css">

In my example above, no matter which page of my website the request is made, they will only see that the request is
coming from my base origin.

Example 8 - sizes

<link rel="icon" sizes="16x16" type="image/x-icon" href="icon/favicon.ico">
