HTML Textarea Element Demo

Example 1

<textarea></textarea>

This is the default behavior of the text area.
It allows for users to change the size of the textarea input, and is whitespace sensitive for user input.

Example 2

<textarea autocomplete autofocus cols=60 rows=10>Hello</textarea>

Here I've started to change the look and functionality of the textarea a little bit.
You'll notice that as you loaded this page, this box was automatically highlighted, that is thanks to the autofocus attribute I've set as true.

Example 3

<textarea disabled>Hello</textarea>

Although the textarea is disabled, you're still able to prefill in text within the textarea tags.

Example 4

<textarea readonly placeholder="Hello"></textarea>

This textarea is readonly, though there is no visual way for one to tell the box is not editable until it is clicked on.

Example 5

  <textarea dirname="name.dir" form="textareaForm" minlength=5 maxlength=30 name="name" required wrap="soft"></textarea>
  <form id="textareaForm">
      <input type="submit">
  </form>

This monstrous element requires that you put at minimum 5 and maximum 30 characters, and will not preserve linebreaks based on the textbox width.
(User inputted line breaks will be preserved)
