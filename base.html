HTML Base Element Demo

Example 1

<base href="https://mywebsite.com/">

In this example, since the href attribute is set, any link on this document will open with https://mywebsite.com/ as the base of that URL.
NOTE: Notice how there is no closing tag. This element does not take any content (outside of its attributes) so a closing tag is excluded.

Example 2

<base target="_blank">

This example shows that the base element doesn't necessarily require the href attribute, you can use just the target attribute if you so choose.
This example will open any link in the document in a new tab.
NOTE: The base element requires AT LEAST one of these two attributes, target or href. You must use at least one.

Example 3

<base href="https://mywebsite.com/" target="_blank">

And naturally, you can use both attributes together if you so choose.
This will have the combined effect of setting https://mywebsite.com/ as the base URL for any link or document in this document, as well as opening that URL in a new browser tab.
